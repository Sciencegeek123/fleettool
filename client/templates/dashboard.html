Fleet Tool Basics - 2 Steps

This is a basic introduction for how to create and update a fleet.
Note: An emergency fleet creation mechanism will be added for response fleets.
Note: This website is still in development. Features are very limited and subject to change.
Note: PLEASE send suggestions or feedback to Peacekeeper. I can't make it better without feedback.

Step 1: Create the fleet

Do this BEFORE the fleet leaves. You can do it as early as you want! This takes time, and you don't want to do it on a timeline.

  1. Press the "CREATE" button on the navigation panel.
  2. Copy and paste the FCs name into the "FC Character Name" box. It is essential you spell this correctly.
  3. Come up with a fleet name. Try to be relevant and descriptive.
  4. Select the fleet catagory. A description of each catagory is found at the bottom of the CREATE page.
  5. Fill in the fleet description telling us what the OP is about and hoping to achieve.
  6. Fill in the fleet composition with details about what people should be flying. *This is so we can check for SRP.*
  7. Look over everything and hit Submit!

Step 2: Update the fleet

Do this AFTER the fleet leaves. This is meant to be quick copy pasting with some notes for after action reports.

  1. Click "FLEETS" on the nav bar and click "UPDATE" next to the fleet you want to add a checkpoint (PAP) to.
         - If you just created a fleet, you will already be on this page.
  2. Copy the fleet member list by doing "ctrl-a ctrl-c" after selecting a name in the fleet chat member list.
  3. Paste the fleet members into the "New Fleet Members" text box.
  4. Optionally: Add a description of what was going on at this checkpoint.
         - You are strongly encouraged to do this. It will be useful in the future.
  5. Look over the information and press Submit
